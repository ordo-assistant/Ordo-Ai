## API Reference
- [All Time Trades (Single)](https://docs.birdeye.so/reference/get-defi-v3-all-time-trades-single.md): Get all time trades or follow duration transactions for one token.
- [All Time Trades (Multiple)](https://docs.birdeye.so/reference/post-defi-v3-all-time-trades-multiple.md): Get all time trades or follow duration transactions for multiple tokens. Max to 20 tokens
- [Wallet - Token Balance (Beta)](https://docs.birdeye.so/reference/get-v1-wallet-token_balance.md): Retrieve the balance of a token in a wallet.
- [Wallet - Balance Change](https://docs.birdeye.so/reference/get-wallet-v2-balance-change.md): Retrieve the balance change history of a wallet.
- [Token - Transfer Total](https://docs.birdeye.so/reference/post-token-v1-transfer-total.md): Return a total number of token transfer transactions for a specified token with various filters like time range, amount, value or source/destination wallet
- [Token - Transfer List](https://docs.birdeye.so/reference/post-token-v1-transfer.md): Return a list of token transfer transactions for a specified token. Provide standardized transaction data such as sender, receiver, amount, value, timestamp, and transaction hash
- [Wallet - Tokens Balance](https://docs.birdeye.so/reference/post-wallet-v2-token-balance.md): Retrieve the balances of a list of tokens in a wallet.
- [Wallet - Transfer total](https://docs.birdeye.so/reference/post-wallet-v2-transfer-total.md): Retrieve total transfer of the wallet.
- [Wallet - Transfer](https://docs.birdeye.so/reference/post-wallet-v2-transfer.md): Retrieve list transfer of the wallet.
- [Supported Networks](https://docs.birdeye.so/reference/get-defi-networks.md): Retrieve a list of all supported networks.
- [Trades - Latest Block Number](https://docs.birdeye.so/reference/get-defi-v3-txs-latest-block.md): Retrieve the latest block number of trades on a chain
- [Token - Creation Token Info](https://docs.birdeye.so/reference/get-defi-token_creation_info.md): Retrieve the creation transaction information of a specified token
- [Token - Trending List](https://docs.birdeye.so/reference/get-defi-token_trending.md): Retrieve a dynamic and up-to-date list of trending tokens based on specified sorting criteria.
- [Token - Holder](https://docs.birdeye.so/reference/get-defi-v3-token-holder.md): Retrieve a list of top holders of a specified token.
- [Token - Holder Distribution](https://docs.birdeye.so/reference/get-holder-v1-distribution.md): Retrieve distribution statistics for token holders based on their share of the total supply. Set include_list=true to return the wallet list for the specified range.
- [Token - Holder (Batch)](https://docs.birdeye.so/reference/post-token-v1-holder-batch.md): Retrieve the token balance held by a list of wallet owners.
- [Meme Token Detail - Single](https://docs.birdeye.so/reference/get-defi-v3-token-meme-detail-single.md): Get detail of given meme token
- [Meme Token - List](https://docs.birdeye.so/reference/get-defi-v3-token-meme-list.md): Get list of meme tokens
- [Price - Historical by unix time](https://docs.birdeye.so/reference/get-defi-historical_price_unix.md): Retrieve historical price by unix timestamp.
- [Price - Historical](https://docs.birdeye.so/reference/get-defi-history_price.md): Retrieve historical price line chart of a specified token.
- [Price - Multiple](https://docs.birdeye.so/reference/get-defi-multi_price.md): Retrieve the latest price information for multiple tokens. Maximum 100 tokens
- [OHLCV - Base/Quote](https://docs.birdeye.so/reference/get-defi-ohlcv-base_quote.md): Retrieve candlestick data in OHLCV format of a specified base-quote pair. Maximum 1000 records.
- [OHLCV - Pair](https://docs.birdeye.so/reference/get-defi-ohlcv-pair.md): Retrieve candlestick data in OHLCV format of a specified pair. Maximum 1000 records.
- [OHLCV](https://docs.birdeye.so/reference/get-defi-ohlcv.md): Retrieve candlestick data in OHLCV format of a specified token. Maximum 1000 records.
- [Price Volume - Single](https://docs.birdeye.so/reference/get-defi-price_volume-single.md): Retrieve price and volume updates of a specified token.
- [Price](https://docs.birdeye.so/reference/get-defi-price.md): Retrieve the latest price information for a specified token.
- [OHLCV V3 - Pair](https://docs.birdeye.so/reference/get-defi-v3-ohlcv-pair.md): Retrieve candlestick data in OHLCV format of a specified pair. Maximum 5000 records. Compared to v1: Added 1s, 15s, and 30s intervals and no candle padding
- [OHLCV V3](https://docs.birdeye.so/reference/get-defi-v3-ohlcv.md): Retrieve candlestick data in OHLCV format of a specified token. Maximum 5000 records. Compared to v1: Added 1s, 15s, and 30s intervals and no candle padding
- [Price - Multiple](https://docs.birdeye.so/reference/post-defi-multi_price.md): Retrieve the latest price information for multiple tokens. Maximum 100 tokens
- [Price Volume - Multi](https://docs.birdeye.so/reference/post-defi-price_volume-multi.md): Retrieve price and volume updates of multiple tokens. Maximum 50 tokens.
- [Search - Token, market Data](https://docs.birdeye.so/reference/get-defi-v3-search.md): Search for tokens and market data by providing a name, symbol, token address, or market address.
- [Utils - Credits Usage of current account](https://docs.birdeye.so/reference/get-utils-v1-credits.md): Retrieve credit usage of current account.
- [Token - Security](https://docs.birdeye.so/reference/get-defi-token_security.md): Retrieve security information of a specified token.
- [Token - Overview](https://docs.birdeye.so/reference/get-defi-token_overview.md): Retrieve stats of a specified token.
- [Pair - Pair Overview (Multiple)](https://docs.birdeye.so/reference/get-defi-v3-pair-overview-multiple.md): Retrieve stats of multiple pairs. Maximum 20 addresses.
- [Pair - Pair Overview (Single)](https://docs.birdeye.so/reference/get-defi-v3-pair-overview-single.md): Retrieve stats of a specified pair.
- [Price stats (Single)](https://docs.birdeye.so/reference/get-defi-v3-price-stats-single.md): Get the price stats (current price, high/low, percentage change) by time frame for one token
- [Token - Liquidity (Multiple)](https://docs.birdeye.so/reference/get-defi-v3-token-exit-liquidity-multiple.md): Provides multiple tokens' liquidity value based on USD, native tokens, and other high-liquidity assets to confirm on-chain pools have real liquidity prior to user trades. Maximum 50 addresses.
- [Token - Liquidity (Single)](https://docs.birdeye.so/reference/get-defi-v3-token-exit-liquidity.md): Provides a liquidity value based on USD, native tokens, and other high-liquidity assets to confirm on-chain pools have real liquidity prior to user trades.
- [Token - Market Data (Multiple)](https://docs.birdeye.so/reference/get-defi-v3-token-market-data-multiple.md): Retrieve market data of multiple tokens. Maximum 20 addresses.
- [Token - Market Data (Single)](https://docs.birdeye.so/reference/get-defi-v3-token-market-data.md): Retrieve market data of a specified token.
- [Token - Metadata (Multiple)](https://docs.birdeye.so/reference/get-defi-v3-token-meta-data-multiple.md): Retrieve metadata of multiple tokens. Maximum 50 addresses.
- [Token - Metadata (Single)](https://docs.birdeye.so/reference/get-defi-v3-token-meta-data-single.md): Retrieve metadata of a specified token.
- [Token - Trade Data (Multiple)](https://docs.birdeye.so/reference/get-defi-v3-token-trade-data-multiple.md): Retrieve trade data of multiple tokens. Maximum 20 addresses.
